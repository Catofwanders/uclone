Server for uclone. GraphQL, Typescrypt, NodeJS.

## How to start

## Server

### 1 `yarn install`

### 2 install on your device postgress database

### 3 Setup .env file with the following data:

- `DB_ENDPOINT=localhost`
- `DB_USERNAME=name`
- `DB_PASSWORD=pass`
- `TWILIO_TOKEN=`
- `TWILIO_SID=`
- `TWILIO_PHONE=`
- `JWT_TOKEN=`
- `yarn run dev`
